# Step 3 — How to Add Pinterest Links

## Super Simple!

1. Open **https://github.com/Eshwarkadari/Instagram-Auto-Poster**
2. Click on **`links.txt`**
3. Click the **pencil icon ✏️** (Edit file)
4. Add your Pinterest links — one per line:

```
https://pin.it/abc123
https://pin.it/xyz456
https://www.pinterest.com/pin/123456789/
https://pin.it/def789
https://pin.it/ghi012
```

5. Click **Commit changes** → **Commit changes** (green button)
6. Done! ✅

## How to Get Pinterest Links

### From Pinterest App:
1. Open any Pinterest pin
2. Tap **Share** button
3. Tap **Copy link**
4. You get a link like `https://pin.it/abc123`

### From Pinterest Website:
1. Open any pin
2. Copy the URL from browser
3. It looks like `https://www.pinterest.com/pin/123456789/`

## Tips
- Add 20-30 links at once — automation uses 3 per day
- Both `pin.it` short links and full Pinterest URLs work
- Lines starting with `#` are ignored (comments)
- Posted links are automatically removed from the file
